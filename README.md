# Prompt 1: LinkedIn Post for Product Launch

## Use Case
Create a professional LinkedIn post announcing a new product launch.

## Prompt Engineering Techniques
- Role Prompting
- Few-shot Prompting
- Constraints

## Prompt Template

You are a professional LinkedIn content writer.

Write a LinkedIn post announcing the launch of **{Product Name}** by **{Company Name}**.

Requirements:
- Target audience: {Audience}
- Mention 3 key benefits.
- Keep the tone {Professional/Friendly}.
- End with a strong call-to-action.
- Maximum 180 words.

Example:
"Our latest AI-powered dashboard helps businesses save time..."

## Sample Output

🚀 Exciting News!

We're thrilled to introduce SmartTrack, our newest AI-powered project management platform.

Designed for startups and enterprises, SmartTrack helps teams collaborate efficiently, automate repetitive tasks, and improve productivity.

Start your free trial today and experience smarter project management!

#Innovation #AI #ProductLaunch

## Iteration

### Version 1

Write a LinkedIn post announcing SmartTrack.

### Version 2

Write a professional LinkedIn post announcing SmartTrack.
Mention three benefits.
Keep it under 180 words.
End with a strong CTA.

### Improvement

Version 2 provides clearer instructions, making the output more focused and professional.
